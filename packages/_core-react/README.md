# @s-8 Core

The `@s-8/core-react` is a conversion of `@s-8/core` for use with react.

## Installation

You can install this library via npm:

```bash
npm install @s-8/core-react
// or
pnpm i @s-8/core-react
// or 
yarn add @s-8/core-react
```
